# document_epic_extract
Gateway de extracción documentaluna, código abierto y escalable. El concepto central busca unificar la extracción de datos visuales creando una capa de abstracción (el meta-gateway en Docker) que trate por igual a los OCR tradicionales (AWS, GCP, Azure), a los modelos locales (SmolVLM2) y a la extensa lista de LLMs comerciales y de código abierto
